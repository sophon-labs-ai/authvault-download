AuthVault

一个安全的离线密码管理器，用 Python 构建。

所有密码使用 AES-256-GCM 本地加密
数据永不离开你的设备，不联网传输
支持 CSV 备份/恢复
完全开源、无广告、无追踪、无需账号

下载 AuthVault-0.0.9.0.zip（Windows）

下载 AuthVault-0.0.9.0.apk（Android）


安装说明（Windows 10/11）

  1. 下载并解压 ZIP 文件到任意文件夹（例如桌面或文档）。
  2. 找到解压后的 AuthVault.exe，右键点击它。
  3. 选择 “发送到” → “桌面（创建快捷方式）”，这样桌面就会出现快捷图标。
  4. 双击桌面快捷方式即可运行 AuthVault。
  5. 程序完全离线，无需安装，所有数据保存在程序同文件夹内，移动文件夹即可带走。

安装说明（Andriod）

  1. 下载apk文件到download文件夹
  2. 找到AuthVault-0.0.9.0.apk点击它。
  3. 不要点扫描应用，点击应用详情，选择不扫描。
第一次运行请设置密码，请妥善保存您的密码，如果丢失，将没有任何办法破解和访问您的数据。
